Welcome to RepRepApp--the app that replicates GitHub repositories into your own repository! It forkes them actually, but it does not matter. What does matter is that it is absolutely free (both of charge and adds)!

IMPORTANT NOTES

  1. DO NOT FORGET TO LOG OUT ONCE YOU'RE DONE

    This app uses safe authentication to your GitHub profile. But as with many other apps, it keeps the connection with GitHub for some time. If you do not log out explicitly, your session will still be on.

  2. YOU DO NOT NEED TO INSTALL ANYTHING

    You do not have to install the app on premises, just run it in your web-browser. If you do need to install it, see the installation instruction on GitHub.

  3. YOU CANNOT REPLICATE YOUR OWN REPOSITORY TO YOUR (SAME) ACCOUNT

    This app forks a repository. GitHub does not allow its users to fork their own repositories into their respective account. The math is simple.
